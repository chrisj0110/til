# Neogit

[neogit](https://github.com/NeogitOrg/neogit)

Git commands:

- commit - `c` or `:Neogit commit` followed by `c` to go straight to the commit message
    - finish commit - `Q`
- status - `:Neogit`
- stage - `s`
- unstage - `u`
- pull - `p` or `:Neogit pull`
- push - `P` or `:Neogit push`
- log - `ll`
- checkout - `bb`
    - checkout -b - `bc`

Navigation:

- toggle expand - tab
- discard - `x`
- help - `?`
- switch to file - `<cr>`
